日常遇到的问题

1.主页面 A 内由组件 B 构成的部分中有一个组件 C，点击后显示组件 C 的页面。组件 B 的组件 C 的页面为什么可以超出主页面中组件部分的大小？
答：组件 C 页面绝对定位加设置宽高就可以实现。
2.$route和$router 的区别？
答：$router 传参 $route 接收参数 query 方式： 传参: this.$router.push({ path:'/xxx' query:{ id:id } }) 接收参数: this.$route.query.id params 方式: this.$router.push({ name:'xxx' params:{ id:id } }) 接收参数: this.$route.params.id

如何避免写重复的代码？

答：使用v-for循环+渲染解决简单的重复代码，使用组件解决代码量较大的。

为了珍惜自己的时间、提高效率，应该多想办法优化代码，使功能可以被封装再利用，而不是重复写垃圾代码。

为什么有些网站定义字体的时候会在后面加上sans-serif？

定义font-family时,最好在最后加一个sans-serif,如果所列出的字体都不能用，则默认的sans-serif字体能保证调用;

给坐标轴及系列设置默认的值 以保证当接入数据不存在时 仍旧可以显示而不是什么都没有
